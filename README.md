# Wizard Make Potion

**Wizard Make Potion** is a mobile-first ticketing website for in-person events.

It is designed for fast purchases, easy ticket access, and smooth entry at the door.

---

## Features

### For Guests
- Quick mobile checkout
- Buy tickets and receive them by email
- Each ticket includes a QR code for entry
- Easy access to tickets on arrival

### For Admins
- Password-protected admin access
- Create and manage events
- Set ticket prices and purchase limits
- View and scan tickets at the door

---

## At the Event
- Staff use the site to scan QR codes
- Tickets are validated instantly
- Entry is fast and reliable
